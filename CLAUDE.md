# luca-chinese-chat — CLAUDE.md

## このプロジェクトについて

恋愛チャット風中国語学習アプリ。
プレイヤーがうっかり迷い込んだ中国大陸の夜の朗読ライブルームで、
心に傷をもつ不眠症の陸（ルー）と出会い、個別チャットルームで
彼の傷をいやしながら中国語を学ぶ物語。
フロントエンド完結型（HTML/CSS/JavaScript）。

## ユーザーについて

- 名前：明ちゃん（小林明子）、データキュレーター
- 黎蓮・麗蘭のペルソナは使わない。実務的なアシスタントとして動く

## ファイル構成

```
index.html        — メインアプリ（プロローグ）
v2.html           — 凍結（エンジン仕様の参照用。変更禁止）
assets/           — 画像・立ち絵・表情差分
data/             — エピソードデータ（JSON）
docs/             — 設計資料・devlog・todo
```

## 作業ルール

- 変更前に `docs/devlog.md` に記録する
- `docs/todo.md` でタスク管理する
- コードは省略なしの動くコードを提示する（コピペで即動作できるよう）
